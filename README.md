Actividad: Condicionales en JavaScript


## Parte A: Operador Ternario

Ejemplo:

Responde o completa el código según el enunciado.

1. Declará una variable numero = 10.
Usá el ternario para mostrar "Par" si el número es divisible por 2, o "Impar" si no lo es.

2. Creá una variable usuarioActivo = true.
Usá el ternario para guardar en otra variable estado el texto "Conectado" si está activo, o
"Desconectado" si no lo está.

3. Dado nota = 7, usá el ternario para mostrar "Aprobado" si la nota es 6 o más, o
"Desaprobado" en caso contrario.

4. Dado dia = "domingo", usá el ternario para mostrar "Día de descanso" si es "sábado" o
"domingo", o "Día laboral" si no.

5. Completá este código:

- let edad = 12;
- let acceso = (edad >= 13)? : ;
- console.log(acceso);
Debe mostrar "Permitido" si la edad es 13 o más, o "Restringido" si no.

## Parte B: Instrucción switch

Ejercicios

1. Dado let dia = "lunes", escribí un switch que muestre:
 “Inicio de semana” si es lunes,
 “Mitad de semana” si es miércoles,
 “Fin de semana” si es sábado o domingo,
 y “Día común” para los demás.

2. Creá un switch con la variable color = "verde" que muestre:
 “Sigue” si es verde,
 “Precaución” si es amarillo,
 “Detente” si es rojo,
 y “Color desconocido” si no coincide.

3. Usá switch con mes = "enero" para mostrar cuántos días tiene el mes (31, 30 o 28/29).

4. Creá un switch con nota = "B" que muestre:
 “Excelente” si es "A",
 “Bueno” si es "B",
 “Suficiente” si es "C",
 y “Reprobado” si no coincide.

6. Dado opcion = 3, hacé un menú simple con switch que muestre:
 “Inicio” si es 1,
 “Configuración” si es 2,
 “Salir” si es 3,
 y “Opción no válida” en default
